📏 Project: Distance Meter
🧰 Use these from your kit:
Arduino Uno
Ultrasonic sensor (HC-SR04 — the one with 2 “eyes”)
16x2 LCD with I2C (the green display with 4 pins)
Breadboard + wires
🔌 Wiring (VERY SIMPLE)
📡 Ultrasonic Sensor (HC-SR04)
VCC → 5V
GND → GND
TRIG → Pin 9
ECHO → Pin 10
🖥️ LCD (I2C – only 4 wires)
GND → GND
VCC → 5V
SDA → A4
SCL → A5
👉 (On Arduino Uno: A4 = SDA, A5 = SCL)
